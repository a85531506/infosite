# StockPulse

A TradingView-inspired stock search and news website. Search for any stock ticker to get real-time quotes and the latest company news — no API key required.

## Features

- Stock symbol search with autocomplete (via Yahoo Finance)
- Real-time quotes (price, change, high, low, volume)
- Latest company news (via Google News)
- Dark theme inspired by TradingView
- Responsive layout
- No API keys or accounts needed

## Usage

Open `index.html` in your browser and search for any stock symbol (e.g., AAPL, TSLA, NVDA).

## Data Sources

- **Stock quotes & search:** Yahoo Finance (public endpoints)
- **News:** Google News RSS
- **CORS proxy:** allorigins.win (for browser-side access)
